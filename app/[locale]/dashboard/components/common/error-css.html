南极冰盖对轨道周期的响应：上新世研究发现

ANDRILL钻探计划揭示的冰川周期性变化
  • 研究发现南极西部冰盖在上新世时期（约5-3百万年前）存在约40,000年周期的冰盖范围变化，与地球轴倾斜度（斜率）变化周期相关联
  • 当行星温度比现在高约3°C，大气CO₂浓度高达约400ppm时，西南极冰盖周期性崩塌，罗斯湾从陆地冰或冰架转变为开放水域
  • 新的冰盖/冰架模型模拟显示，受斜率影响的海洋诱导融化可导致南极冰量波动，相当于海平面上升最多+7米（西南极）和+3米（东南极）
  • 间冰期期间，硅藻沉积物表明表层水生产力高，夏季海冰最少，气温高于冰点，表明在CO₂浓度升高的条件下表面融化的额外影响
